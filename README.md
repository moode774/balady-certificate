# Balady Health Certificate

موقع عرض الشهادة الصحية - نظام بلدي

## الملفات

- `index.html` - الصفحة الرئيسية (عرض البيانات)
- `certificate.html` - صفحة الطباعة (PDF)
- `logo(1).svg` - شعار بلدي
- `Vision 2030.jpg` - شعار رؤية 2030
- `Y.jpg` - الصورة الشخصية

## كيفية الاستخدام

1. افتح `index.html` لعرض البيانات
2. اضغط على القائمة (☰) ثم "طباعة الشهادة"
3. في صفحة الشهادة، اضغط "طباعة / حفظ PDF"
